## Sobre mim
Olá, sou um desenvolvedor full stack com foco em Java no backend e AngularJS no frontend. <br>
Desde o primeiro dia, tenho-me dedicado ao estudo para dominar todo o ciclo de desenvolvimento de serviços web. Ainda tenho muito a aprender, mas sou apaixonado pelo que faço. <br>
A estrada da aprendizagem é interminável, e estou imerso nela com entusiasmo. A cada desafio, encontro uma oportunidade de crescimento. Estou ansioso por descobrir mais neste percurso fascinante.
